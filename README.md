# Daily 学习样式 Starter Kit

这是一个可复制的 Obsidian 学习系统样式包。它只包含结构、规则、模板和示例，不绑定具体专业、课程或个人进度。

## 复制方式

1. 把整个 `starter-kit/` 目录复制到你的 Obsidian Vault。
2. 把目录名改成你想要的名字，例如 `daily`、`study` 或 `learning-system`。
3. 打开 `00-入口/学习系统导航.md`。
4. 按 `00-入口/当前学习上下文.md` 填入你的主线、当前阶段、今日焦点。
5. 如果使用 Templater，把 `_templater/` 加入 Templater 的模板目录。

## 初始化顺序

1. 填 `00-入口/当前学习上下文.md`。
2. 填 `02-规划规则/每日学习规划要求.md`。
3. 按你的方向改 `02-规划规则/笔记与练习题样式约定.md`。
4. 新建第一周：复制 `03-周记录/_示例周/`，改名为 `第1周`。
5. 新建第一天：复制 `03-周记录/_示例周/_示例日期/`，改名为 `MM-DD`。

## 设计原则

- 入口固定：每天先看当前上下文和复习队列。
- 主线收敛：一个阶段只保留一个主要学习方向。
- 产出驱动：学习必须落到笔记、练习、复盘或知识点卡片。
- 滚动规划：周计划只细排最近 1-2 天，后续按完成情况更新。
- AI 可接手：把规则、上下文、模板和边界写进固定位置。

## 目录说明

- `00-入口/`：日常入口、当前上下文、复习队列。
- `01-AI协作/`：AI 如何读取上下文、如何写笔记、如何更新进度。
- `02-规划规则/`：规划、笔记、练习题和模板规范。
- `03-周记录/`：每周计划、每日进度、学习笔记和练习题。
- `04-知识点索引/`：沉淀长期可复习的知识点卡片。
- `_templater/`：Obsidian Templater 模板。

## 使用建议

不要一开始就把目录做复杂。先让系统跑通 7 天，再调整模板字段。
